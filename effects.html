Analog Circuit Effects
Moog Ladder Filter

Designed by Robert Moog in 1966, the Moog Voltage-Controlled Filter (VCF)—or Moog ladder filter—is a key element of the Moog synthesizer sound and a classic piece of electronic music history. In addition to its popularity today, the filter found its place on a number of famous records from the 1970s and 80s, including Kraftwerk's The Man Machine, Bob Marley's Catch a Fire, and Michael Jackson's Thriller, to name a few.

The Moog VCF is a set of four low-pass filters connected by 10 transistors. I implemented a linear model of the circuit using the state-space modeling technique. The method leads to a system of differential equations that I solved using three different approaches: forward, backward, and trapezoidal integration. The C++ code for the trapezoidal integration is available to download on my GitHub page. The page also includes instructions for compiling the source code and running the VST plugin in a DAW.

Vacuum Tube Distortion

Vacuum tube distortion is a vintage audio effect found in many old music amplifiers, especially guitar amplifiers. One well-known guitar amplifier that produces this type of distortion is the Fender Bassman (shown at right). The effect is caused by a valve-like clipping that adds additional harmonics to the original input audio.

I modeled the valve-like clipping using an approach based on the work of R. Bendiksen. The main equation of the model takes the form $$y = {x - Q \over 1 - exp(-D(x-Q))} + {Q \over 1 - exp(DQ)},$$ where \(x \) is the original input audio, \(y \) is the distorted output audio, \(Q \) is a parameter that determines the linearity of the clipping function, and \(D \) defines the harshness of the distortion.

Fluid Sonification Effects

Inspired by Iannis Xenakis' 1956 work Pithoprakta, I used a modern laser trapping setup to track Brownian particles and sonify their motion. I built the optics setup as part of my master's thesis in Physics at Wake Forest University in Winston-Salem, North Carolina. I sonified the position data of single, optically-trapped Brownian microspheres using Pure Data and Ableton Live. Each radial displacement of the sphere mapped to an audible pitch.

I used three different mapping schemes—equal-area, biased, and probability mapping. In each scheme, the centermost subregion of the mapping region corresponded to a single pure tone. In equal-area mapping, the note increased along a particular musical scale as the particle moved out radially into different equal-area subregions. In biased mapping, the centermost subregion area was enlarged relative to the other subregions. In probability mapping, the note had a 50-50 percent probability of increasing by some integer number of MIDI note numbers or decreasing by some other integer number of MIDI note numbers each time the particle entered a new subregion. I wrote a conference paper that provides more details on the project. I also made a couple of videos (here and here) that synchronize the sounds to the actual experimental motion.

Other Effects
Phase Vocoder

The phase vocoder is an audio algorithm that stretches or compresses a signal. Sound engineers have used the phase vocoder in a practical way to modify the length of audio signals. Musicians and sound designers have also used the basic principle of the phase vocoder to create pitch-shifting and pitch-correcting software, such as Auto-Tune.

I implemented a frequency-domain version of the phase vocoder that makes use of the short-time Fourier transform, phase wrapping, and overlap adding. Notably, the filtered audio retains the frequency content of the original input signal. That is, no pitch-shifting is observed.

Delay Effects

Delay effects are ubiquitous in modern music production. Common delay effects include comb filtering, flanger, and chorus. I developed algorithms to produce each of these effects. The C++ code for the feedback comb filter is available on my GitHub page.
