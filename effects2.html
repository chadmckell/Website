Room Reverberation

I modeled room reverberation using the image-source method. The model simulates the impulse response of a room and then convolves the impulse response with the original input audio in the time domain. For simplicity, I assumed that the walls of the room were flat and positioned perpendicular to each other.

The equation that defines the magnitude \(g\) of each impulse in the simulated impulse response takes the form $$g = {(\sqrt{1-\alpha})^w \over l}$$ where \(\alpha\) is the absorption coefficient of the walls, \(w\) is the total number of collisions between the sound wave and the walls for each image source, and \(l\) is the total distance between the receiver and each image source. (Note that high absorption values correspond to low reverberant rooms). The MATLAB code for my reverb model is available on my GitHub page.

Phase Vocoder

The phase vocoder is an audio algorithm that stretches or compresses a signal. Sound engineers have used the phase vocoder in a practical way to modify the length of audio signals. Musicians and sound designers have also used the basic principle of the phase vocoder to create pitch-shifting and pitch-correcting software, such as Auto-Tune.

I implemented a frequency-domain version of the phase vocoder that makes use of the short-time Fourier transform, phase wrapping, and overlap adding. Notably, the filtered audio retains the frequency content of the original input signal. That is, no pitch-shifting is observed.

Digital Delay

Delay effects are ubiquitous in modern music production. Common delay effects include comb filtering, flanger, and chorus. I developed algorithms to produce each of these effects. The C++ code for the feedback comb filter is available on my GitHub page.
